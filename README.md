# Customer Purchase Behavior Analysis

## Objective
Analyze large-scale retail transaction data to understand customer purchasing behavior, identify high-value customers and discover sales trends.

## Dataset
The analysis uses the Online Retail II dataset containing over 500,000 transaction records.
Due to file size limitations, the dataset is not included in this repository.

You can download it from Kaggle:
- [Online Retail II Dataset](https://www.kaggle.com/datasets/lakshmi25npathi/online-retail-dataset?resource=download)

After downloading, place the Excel file in the `archive/` folder.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- OpenPyXL

## Key Insights
- Customer spending shows a long-tailed distribution
- Top 10% of customers contribute a large portion of total revenue
- Sales exhibit seasonal trends
- A small number of countries dominate total revenue

## Business Impact
This analysis helps businesses identify high-value customers, optimize inventory planning and design targeted marketing strategies.
